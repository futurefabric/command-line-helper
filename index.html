  • Show hidden files defaults write com.apple.finder AppleShowAllFiles TRUE
    killall Finder
    Hide hidden files defaults write com.apple.finder AppleShowAllFiles FALSE
    killall Finder
    Show current location pwd
    Edit file pico [filename]
    Create folder alias/shortcut cd /Users/[Username]
    pico .bashrc
    alias [aliasname]='/absolute/path/to/folder'

    . ~/.bashrc
    Show and kill running ruby processes ps axf | grep ruby
    kill -9 0000
    Change folder permissions eg. chmod -R 777 (recursive) or chmod 777 folder (for specific folder)
    Screencapture at an interval while [ 1 ];do vardate=$(date +%d\-%m\-%Y\_%H.%M.%S); screencapture -t jpg -x ~/Desktop/Folder/$vardate.jpg; sleep 10; done
    (where Folder can be specified and the interval is set in seconds after sleep)
  • Current repo status git status
    Pull, rebase git pull rebase
    Commit git commit -am "[Commit Message]"
    Add a remote git remote add github git@github.com:example/foo.git
    Show remotes git remote
    Push to a remote git push [Remote name] [Branch name] (eg. git push origin master)
    Merge conflict, use their file git checkout MERGE_HEAD -- path/to/file.filetype
    Get branch from remote and checkout locally git checkout -b branch_name origin/branch_name
    Cherry pick a commit from another branch git cherry-pick -n 428366cf7295568f2363342a5d33016f57422058 (you then need to commit change)
    Delete most recent commit git reset --hard HEAD~1
    Unstage most recent commit git reset HEAD^ (append number to unstage more thatn one commit)
    Delete local branch git branch -D branchname
    Pull remote branch (well, create local branch and track remote) git checkout --track origin/branch_name
    Push a local branch to heroku as master git push heroku yourbranch:master
